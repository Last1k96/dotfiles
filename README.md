# Dotfiles

Personal development environment for Ubuntu.

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/Last1k96/dotfiles/main/install.sh | bash
```

After installation, log out and back in for zsh to take effect.

### Tmux plugins

On first tmux launch, press `prefix + I` (Ctrl-A then Shift-I) to install plugins via TPM.
